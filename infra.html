on-prem [homelab hardware]:

dell optiplex 3070 - main hypervisor that hosts all my homelab vm's and containers, it is the part of the ProxMox cluster together with the lenovo m900 and raspberry pi [pi is used as a q device]. It has i7-8700T CPU, 32GB RAM and 1TB SSD and I use it as a production server with configured replication to a second cluster member.

lenovo thinkcentre m900 - staging hypervisor that hosts staging part of my homelab vm's and containers, part of ProxMox cluster, has i5-6500T, 32GB RAM I use it mostly for testing. It replicates VMs from a production cluster member.

protectli vp2420 - main firewall [opnsense] that I use right after the ISP DSL. The firewall is configured to handle VLANs, IPS/IDS, DHCP, DNS upstream, NTP, VPN and firewalling. I am running Zenarmor NextGen firewall as an additional feature. Protectli BIOS is flashed with the Seabios open source firmware.

cisco sg200-26 - main network switch

netgear wax220 - WiFi access points, with OpenWRT firmware.

qnap ts-453-a - NAS that I use to backup everything.

workstations:

intel nuc kit - main workstation with Fedora Linux

lenovo thinkpad x230 - my main laptop with Fedora Linux, mostly used when traveling or working remotely.

macbook pro 15" 2018 - main computer for audio & video editing and music production. I do all Ableton and modular stuff here.
